# 🐻 Chicago Bears Depth Chart Viewer

A clean, responsive web app showing the current Chicago Bears offensive and defensive depth chart for the 2025–26 season.

## Features

- 📋 Full offensive depth chart (QB, RB, OL, WR, TE)
- 🛡️ Full defensive depth chart (DL, LB, Secondary)
- ⚡ Special teams (K, P, LS, returners)
- 🎓 Coaching staff + key FA moves
- Collapsible position rows
- Mobile-friendly responsive layout
- Bears-themed dark design (navy + orange)

## Running Locally

Just open `public/index.html` in a browser — it's a static HTML file, no server needed.

Or serve with any static server:

```bash
npx serve public
```

## Data Source

Depth chart data sourced from [ourlads.com](https://www.ourlads.com/nfldepthcharts/depthchart/CHI) and [CBS Sports](https://www.cbssports.com/nfl/teams/chi/chicago-bears/depth-chart/), last updated March 14, 2026.

## Stack

Pure HTML/CSS/JS — no dependencies, no build step, no framework. Intentionally simple.
